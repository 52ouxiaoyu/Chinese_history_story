# 09-汉朝的“推恩令”

在中国历史上，皇帝与地方诸侯王之间的权力斗争，贯穿了许多朝代。皇帝，希望中央集权，把所有权力都牢牢抓在自己手里；而诸侯王，则希望守住自己的一亩三分地，甚至还想扩大自己的势力。

在汉朝初期，这个问题尤其严重。当时的诸侯王，大多是刘姓宗室，他们拥有自己的土地、军队，甚至可以自己收税、铸钱。他们就像一个个“国中之国”，对皇帝的统治构成了巨大的威胁。汉景帝时期，就曾因为削减诸侯王的封地，而引发了“七国之乱”，险些动摇了汉朝的根基。

到了汉武帝刘彻即位时，他面对的，就是这样一个棘手的局面。他知道，这些强大的诸侯王，就像一颗颗定时炸弹，必须要想办法拆除。

直接用武力削藩？风险太大，可能会重蹈“七国之乱”的覆辙。用法律制裁他们？他们都是皇亲国戚，盘根错节，很难找到下手的机会。

就在汉武帝一筹莫展的时候，一位名叫主父偃（读音：zhǔ fù yǎn）的大臣，给他出了一个主意。这个主意，堪称中国历史上最“温柔”，也最“阴险”的阳谋。它就像一杯甜美的毒酒，让诸侯王们笑着喝下去，然后，在不知不觉中，被瓦解得干干净净。

这个绝妙的计策，就是——“推恩令”。

“推恩令”，从字面上看，充满了善意。“推”，就是推广；“恩”，就是恩惠。意思是，皇帝要将自己的恩惠，推广给更多的人。

听起来，像不像一个充满仁慈的德政？

让我们来看看，“推恩令”的具体内容是什么。

在推恩令颁布之前，汉朝的诸侯王，和周朝一样，实行的是“嫡长子继承制”。也就是说，老诸侯王去世后，他所有的封地和爵位，都由正妻生的大儿子，也就是“嫡长子”，一个人来继承。

其他的儿子们，被称为“庶子”，他们虽然也是王室血脉，但却分不到一寸土地，只能领一点零花钱，当个富贵闲人。

你可以想象，在这样的制度下，那些庶子们的心里，该有多么不平衡。同样是父王的孩子，凭什么大哥可以拥有一切，而我们却什么都没有？这种不满，就像一颗种子，埋在每个诸侯王国的内部。

主父偃，就敏锐地抓住了这个矛盾。他对汉武帝说：“陛下，如今的诸侯王，封地动辄方圆几百里。他们只让嫡长子一人继承，其他的儿子们，都无尺寸之地。这既不符合我们孝顺仁义的古训，也容易让他们心生怨恨，引发内乱。臣请求陛下，下一道命令，允许诸侯王们‘推私恩’，将自己的封地，分给所有的儿子们，让他们都能成为侯爵。”

汉武帝一听，立刻就明白了这招的厉害之处。

他立刻颁布了“推恩令”，并昭告天下。内容大致是：

“各位诸侯王啊，你们的封地那么大，只有一个儿子继承，实在是太不公平了。朕于心不忍啊！现在，朕给你们一个机会，来体现你们伟大的父爱。你们可以把自己的王国，分出一部分来，封给你们其他的儿子们，让他们也成为列侯。这些新封的侯国，直接归所在地的郡县管辖，地位和我直辖的侯国一样。这样一来，你们所有的儿子，都能享受到你们的恩泽，这不是一件大好事吗？”

这道法令一颁布，诸侯王们，陷入了巨大的困境。

他们反对吗？怎么反对？皇帝说得很清楚，这是让你们“推私恩”，是给你们机会去爱护自己的孩子。你要是反对，那你就是不慈爱的父亲，你就是自私自利，你就是不体恤自己的亲骨肉。这在以“孝”治天下的汉朝，是从道德上就站不住脚的。

而且，这道命令，不是强制的，是“允许”你们这么做。做不做，是你们自己的事。

更要命的是，诸侯王们还没来得及反对，他们家里的那些庶子们，先欢呼雀跃起来了。

他们一个个跑到自己的父王面前，哭着喊着，请求父王“推恩”。

“父王啊！您看，皇帝陛下都这么仁慈，您就分我一小块地吧！”
“是啊，父王，我不要多，只要一个小小的县城，能让我当个侯爵，我就心满意足了！”

面对儿子们的“亲情攻势”，诸侯王们，根本无法拒绝。手心手背都是肉，哪个父亲，不希望自己的孩子都有个好前程呢？更何况，这背后还有皇帝的“鼓励”和整个社会舆论的压力。

于是，诸侯王们，只能“自愿”地，开始分割自己的王国。

让我们来想象一下这个过程。

比如，一个叫刘大壮的诸侯王，他有一个很大的王国，方圆五百里。他有一个嫡长子，叫刘A，还有三个庶子，叫刘B、刘C、刘D。

按照老的制度，刘大壮死后，这五百里的地，全是刘A的。

但现在有了“推恩令”。刘大壮为了“体现父爱”，也为了安抚几个小儿子，他可能会这么分：

嫡长子刘A，继承王位，以及王国最核心的三百里土地。
庶子刘B，分到东边的一个郡，五十里地，封为“东郡侯”。
庶子刘C，分到西边的一个县，三十里地，封为“西县侯”。
庶子刘D，分到南边的一片山，二十里地，封为“南山侯”。

看起来，皆大-欢喜。刘大壮当了个好父亲，儿子们也都有了自己的封地。

但是，魔鬼，藏在细节里。

首先，这个曾经方圆五百里的大王国，一下子就被分成了四块。最大的那一块，也只有三百里了。

其次，也是最关键的一点：法令规定，这些新封的“侯国”，虽然名义上是从王国里分出去的，但它们不再归新的诸侯王（刘A）管辖，而是直接由汉朝中央政府派出的郡守来管理。

这意味着什么？这意味着，刘B、刘C、刘D的封地，实际上，已经从“诸侯王国”的版图里，被割走，变成了汉武帝的“直辖领地”。

这个过程，就像切蛋糕。

原来的诸侯王国，是一块完整的大蛋糕。推恩令，就是一把刀。它不是由皇帝来切，而是让诸侯王自己，把自己的蛋糕，切成一小块一小块，分给自己的儿子们。而这些切出去的小蛋糕，立刻就被皇帝，端到了自己的盘子里。

一代人之后，这个王国，就从五百里，变成了三百里。

那么，两代人之后呢？

新的诸侯王刘A，他也有好几个儿子。他也要“推恩”，也要分。于是，他那三百里的地，可能又被分成了几块，变成了两百里，一百里……

就这样，一代传一代，曾经强大无比的诸侯王国，被不断地分割，不断地缩小。就像一个大冰块，放在太阳底下，自己慢慢地融化，蒸发，最后，只剩下小小的一块冰核。

到了最后，这些所谓的“诸侯王”，他们所拥有的土地，可能只有一个县那么大，甚至更小。他们空有一个“王”的头衔，但实力，和一个普通的富家翁，已经没什么区别了。他们再也无力，也无心，去对抗强大的中央政府。

汉武帝，这位雄才大略的皇帝，不费一兵一卒，不流一滴血，就用这样一道看似温情脉脉的“推恩令”，兵不血刃地解决了困扰汉朝数十年的心腹大患。

这，就是“阳谋”的最高境界。它不像阴谋那样，需要偷偷摸摸。它把所有的规则都摆在桌面上，光明正大。它利用的，不是刀剑，而是人性中最根本的弱点——亲情、贪婪和虚荣。它让你明知道这是个陷阱，却又心甘情愿地，一步步走了进去。

从此以后，汉朝的中央集权，得到了空前的加强，一个强大、统一的帝国，真正地屹立在了世界的东方。而那道“推恩令”，也作为历史上最经典的政治谋略之一，被后人反复研究和赞叹。
