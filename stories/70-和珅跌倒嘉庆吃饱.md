
# “和珅跌倒，嘉庆吃饱”：史上第一大贪官，究竟积攒了多少财富？

在中国历史上，提起“贪官”，有一个人的名字，几乎是家喻户晓，他就是清朝乾隆年间的权臣——和珅。

一句“和珅跌倒，嘉庆吃饱”的民间谚语，流传了数百年，生动地描绘了这位“史上第一大贪官”，所聚敛的财富，是何等的惊人。

那么，和珅，这位深受乾隆皇帝宠信，权倾朝野二十余年的人物，他究竟贪了多少钱？他的财富，又是如何积累起来的呢？

关于和珅财富的确切数字，史学界并没有一个统一的定论，因为其财产，并没有被完全、精确地统计过。但从嘉庆皇帝，在扳倒和珅后，所公布的抄家清单中，我们可以窥见其财富的冰山一角。

这份清单，名为《和珅犯罪二十大款》，其中列举的财产，令人瞠目结舌。

**我们先来看看他的不动产：**

*   **府邸与园林**：和珅的府邸，就是今天位于北京恭王府。这座府邸的规模，在当时，是逾制（超越规矩）的。其内部的装饰，使用了本应只有皇宫才能使用的楠木和金丝楠木。此外，他在京城内外，还拥有大量的私家园林和别墅。
*   **土地**：被查抄的土地，多达8000余顷，也就是80多万亩。这是一个什么概念？清朝一个普通的自耕农，能拥有三五十亩土地，就已经算是小康之家了。和珅拥有的土地，相当于数万个农民家庭的土地总和。
*   **房产**：在北京、天津、通州等地，被查抄的房产，多达3000多间。这些房产，大多位于当时最繁华的商业地段。

**再来看看他的动产，那就更加夸张了：**

*   **金银现金**：抄出的现银，有8000万至1亿两白银之间。而当时，清政府每年的财政总收入，也不过七八千万两白银。也就是说，和珅一个人贪污的财富，就相当于当时大清国，一年半甚至两年的国家财政收入总和。这，就是“和珅跌倒，嘉庆吃饱”这句谚语，最直接的来源。
*   **珍奇异宝**：
    *   **黄金**：抄出的黄金，数量巨大，包括金元宝、金条、金器皿等，折合白银，约有数千万两。
    *   **珠宝玉器**：直径超过一寸的大珍珠，有数十颗；比拳头还大的红宝石、蓝宝石、猫眼石，不计其数。整块的玉山、玉如意、玉器古玩，更是堆满了几个库房。
    *   **衣物皮货**：仅貂皮大衣，就有800多件；各种绫罗绸缎、名贵皮毛，多达数万匹。
*   **商业资产**：和珅不仅贪污，他还非常善于“理财”。他在全国各地，开设了大量的当铺、银号、矿山、盐号、古玩店等。这些产业，每年都能为他带来巨额的利润。他建立了一个庞大的“商业帝国”，形成了一个“贪腐-洗钱-投资-再生财”的完整产业链。

综合估算，和珅的总资产，大约在8亿至11亿两白银之间。这个数字，相当于清政府15年的财政收入。如果换算成今天的购买力，这是一个难以想象的天文数字。当时，一位西方的传教士，在目睹了和珅的财富后，惊叹道：“他所拥有的财富，是全世界都罕见的。”

那么，和珅是如何在短短二十几年里，聚敛起如此惊人的财富的呢？他的贪腐手段，可谓是“集大成者”，五花八门，无所不用其极。

1.  **公然索贿，卖官鬻爵**：作为军机大臣、领班大学士，和珅掌握着朝廷的人事任免大权。无论是地方官的升迁，还是京官的调补，都必须经过他的手。于是，“和珅门下走，银子是敲门砖”，成了当时官场公开的秘密。一个职位，从几千两到几万两白天，被他标好了价码，公开出售。

2.  **把持朝政，贪污国帑**：和珅身兼数职，户部（财政）、吏部（人事）、工部（工程）等要害部门，都由他实际掌控。国家的重大工程，如修缮宫殿、治理河道等，都成了他中饱私囊的绝佳机会。他通过虚报预算、克扣工料等方式，将大量的国家拨款，装进了自己的腰包。

3.  **巧立名目，敲诈勒索**：和珅非常善于利用乾隆皇帝的名义，来为自己敛财。比如，某地发生灾情，他会以“奉旨赈灾”的名义，向地方富商“劝捐”。这些捐款，大部分，都落入了他的私囊。他还创立了“议罪银”制度，允许犯罪的官员，通过缴纳罚款，来减轻或免除自己的罪行。这项制度，本意是充盈国库，但实际上，却成了和珅为自己合法敛财的工具。

4.  **经营工商，垄断市场**：和珅利用手中的权力，为自己的商业活动，大开绿灯。他开设的银号，可以承接国家的存款业务；他经营的矿山，可以获得优先开采权。他利用信息不对称和权力不对等，在市场上，进行投机倒把，获取暴利。

和珅能够如此肆无忌惮地贪腐，最根本的原因，是他得到了乾隆皇帝，近乎无条件的宠信和纵容。

和珅聪明、能干，尤其善于揣摩乾隆的心思。他不仅是乾隆的臣子，更是他的玩伴、理财顾问和“背锅侠”。乾隆晚年，好大喜功，生活奢靡，需要巨大的开销。和珅总有办法，为皇帝搞到钱，满足他的一切需求，而又不至于让国库，显得太过空虚。在乾隆看来，和珅就是他最听话、最能干的“家奴”。

然而，当乾隆皇帝的生命，走到尽头时，和珅的末日，也就来临了。

乾隆驾崩后的第十五天，嘉庆皇帝，就以雷霆手段，宣布了和珅的“二十大罪状”，并下令将其抄家、赐死。

和珅的倒台，看似突然，实则必然。他的存在，已经严重地威胁到了皇权本身。他的财富，富可敌国；他的权力，遍布朝野。对于新皇帝嘉庆来说，不扳倒和珅，自己的龙椅，就坐不稳。

“和珅跌倒，嘉庆吃饱”，这句谚语，既是对一个巨贪财富的惊叹，也是对封建专制制度下，权臣与皇帝之间，那种畸形共生关系的绝妙讽刺。和珅的悲剧，不仅仅是一个贪官的个人悲剧，更是那个“盛世”背后，制度性腐败的必然产物。
